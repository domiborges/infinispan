= Setting up the Spring Boot Starter

Add dependencies for the {brandname} Spring Boot Starter to your project.

[IMPORTANT]
====
{brandname} supports Spring Boot version 2.x and version 3.
Be aware that Spring Boot version 3 requires Java 17.

The examples in this document include artifacts for the latest version of Spring Boot.
If you want to use Spring Boot 2.x use:

* `infinispan-spring-boot-starter-embedded`
* `infinispan-spring-boot-starter-remote`
====
