= Adding custom roles to Cache Manager configuration

You can customize authorization settings in your {brandname} configuration to assign users different combinations of roles and permissions.

.Procedure

. Open your {brandname} configuration for editing.
. Configure authorization for caches to restrict access based on user roles.
. Assign roles when you create {brandname} users.
